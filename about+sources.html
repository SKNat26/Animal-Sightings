Development

How did this website come to be? It was thought of through the 2022 summer intensive program from All Star Codes. With many loose ends, this project was hastily done in about 5 days for Demo Day. However, the project was later perfected through the 6-week program, Tech Entrepreneurship Incubator, where it was made into a more practical website that would appeal to more people. The frontend was made with simple HTML and CSS and the backend was made with JavaScript, Firebase, and a few APIs.
